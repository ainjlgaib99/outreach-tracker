# Outreach Tracker

Freelance outreach CRM — single-file PWA for tracking leads, proposals, follow-ups, and pipeline status.

## Stack
- Single-file PWA (HTML/CSS/JS)
- IndexedDB (local persistence)
- Airtable sync optional
- Netlify/Vercel deploy

## How to run
Drop `index.html` into Netlify — zero config.
